# N6CoconaLab
 .NET6 + Cocona → CLI  App

# 關鍵說明
`Cocoan` 設計用於開發 CLI App 的框架。讓開發者可以快速專注到系統核心/商業邏輯的開發。
1. 制式化處理 Arguments 的解析與傳遞到對應的 Command。
2. 制式化 Command 的說明輸出。
